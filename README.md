## 🦷 Dental Clinic — SmileCare

A complete, modern dental clinic website with full backend support.

### ✨ Features

- ✅ Fully responsive — mobile first design
- ✅ Appointment booking form → saves to MongoDB
- ✅ Email confirmation to patient via Nodemailer
- ✅ Admin dashboard at `/admin` — view, confirm, cancel appointments
- ✅ Floating WhatsApp CTA button
- ✅ Smooth scroll animations
- ✅ Hindi + English branding (*Aapki Smile, Hamari Zimmedari*)
- ✅ Pure HTML/CSS/JS frontend — no heavy frameworks
- ✅ REST API (Express) — clean and production ready

## 🚀 How to Use

**Step 1 — Clone the repo**
```bash
git clone https://github.com/ashishyadav/indian-freelance-web-templates.git
cd indian-freelance-web-templates/smilecare
```

**Step 2 — Setup environment**
```bash
cp .env.example .env
# Fill in your MongoDB URI and SMTP credentials
```

**Step 3 — Run locally**
```bash
npm install
npm run dev
# Open http://localhost:3000
```
